# CareTrack API

## Problema identificado

As informações da cuidadora estavam espalhadas entre memória, caderno e mensagens.

## Solução proposta

Criar uma API simples para centralizar pacientes e atendimentos.

## Tecnologias

- Node.js
- Express
- SQLite

## Como rodar

npm install
npm start

## Rotas

GET /pacientes
POST /pacientes
GET /atendimentos
POST /atendimentos
GET /historico/:id
